# Emby Migrator v1.1.3

`v1.1.3` 缩短版本检测缓存时间，让新版本提示更及时。

## 改进

- GitHub 最新稳定版本的成功检测缓存从 6 小时缩短为 1 小时。
- 检测失败仍缓存 15 分钟，避免网络异常时反复请求。
- `/api/version` 返回 `Cache-Control: no-store`，避免浏览器缓存旧的检测结果。
- 容器启动后仍会立即执行首次检查；重启容器可立即清空内存缓存并重新检测。

## 验证

- 缓存时效与 HTTP 缓存头回归测试通过。
- 全量 Go 测试、`go vet`、JavaScript 语法、模块完整性和敏感信息检查通过。
