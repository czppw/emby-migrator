# Emby Migrator v1.1.2

`v1.1.2` 完善登录体验和媒体技术信息的产品展示，并增加安全的单用户账号修改能力。

## 新增与改进

- 登录页改为标准账号与密码表单，默认账号为 `admin`，浏览器可识别并保存登录信息。
- 登录账号不再硬编码在输入框中；首次使用回落到 `admin`，之后优先使用当前浏览器最后成功登录或修改后的账号。
- 安全设置改为一个表单统一修改账号与密码，新密码可留空，仅保存实际发生的变化。
- 保存前统一验证当前密码，账号和密码在同一次配置写入中原子更新，完成后立即退出。
- 修改账号后，旧账号和已有登录会话立即失效；同时修改密码时使用新密码重新登录。
- 登录页迁移流程、能力说明和主页标题补充媒体技术信息，明确包含 MediaSources、MediaStreams 与 Chapters。

## 安全与兼容

- 账号长度限制为 3-32 个字符，支持字母、数字、中文以及 `.`、`-`、`_`。
- 现有安装继续使用 `admin` 或已经持久化的账号与密码，不影响 Emby 服务器和媒体库配置。
- 旧版只提交密码的登录请求仍按默认 `admin` 兼容处理。

## 验证

- 单用户账号修改、错误密码、非法账号、旧会话失效、旧账号拒绝和新账号登录测试通过。
- 全量 Go 测试、`go vet`、JavaScript 语法、模块完整性和公开文件敏感信息检查通过。
